CSS 教學 > 顏色

color (顏色) 屬性能讓我們在 CSS 中設定一個元素的顏色。

這個屬性可以有以下三類的設定值：

  • 十六進位值
  • RGB 碼
  • 顏色名稱

color 屬性的語法如下：

十六進位值:

{color:#XXXXXX;}

X 為十六進位碼 (從 0 到 9，以及 A 到 F)。

RGB:

{color:rgb(X,Y,Z); } X、Y、和 Z 都是介於 0 和 255 之間的數字。

或是 {color:rgb(X%,Y%,Z%); } X、Y、和 Z 都是介於 0 和 100 之間的數字。

顏色名稱:

{color:[顏色名稱];}

十六進位值是一個以六碼 (如果前兩碼、中間兩碼、以及最後兩碼都一樣的話，那也可以以三碼的形式表現。舉例來說，#FFF 及 #FFFFFF 都是代表白色) 來設定顏色。這一個十六進位碼顏色表可以讓我們知道什麼碼會對應到什麼顏色。 在這裡，我們就以上三種方式各舉一個例子：

CSS樣式 顯示結果
p {color:#FF0000;} FF0000 代表紅色。
p {color:rgb (255,0,255);} rgb(255,0,255) 是粉紅色。
p {color:green;} 這是綠色。
